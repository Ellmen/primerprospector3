Primer Prospector Tutorial
==========================
The Primer Prospector tutorial illustrates how to use various features of Primer Prospector. We recommend that all users begin with the `Primer Prospector overview tutorial <./tutorial.html>`_ which takes the user through generating `de novo` primers and analysis of existing primers.
